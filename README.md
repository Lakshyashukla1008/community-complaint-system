# 🌟 Yuva Shakti Sangathan App

A Streamlit-based web application for managing community complaints and contributions.

---

## 🚀 Features

- 🔐 User Authentication (Login / Signup)
- 📝 Complaint Submission System
- 💰 Contribution Management
- 📍 Auto Location Detection (IP-based)
- 📧 Email Notifications (User + Admin)
- 🗄️ MongoDB Atlas Integration
- 🔒 Secure Credentials using .env

---

## 🛠️ Tech Stack

- Python (Streamlit)
- MongoDB Atlas
- SMTP (Email Service)
- dotenv (Environment Variables)

---

## 📍 Upcoming Features

- 💳 PhonePe Payment API Integration
- 📌 Map-based Location (Pin system)
- 🛠️ Admin Dashboard (Manage complaints)
- 📊 Complaint Status Tracking
